### 목표 기능
- [ ] 유저 회원가입
- [ ] 로그인
- [ ] 로그아웃
- [ ] 유저 + 저장 이미지 정보조회
- [X] 이미지 업로드
- [X] 이미지 메타 데이터 저장
- [X] 이미지 삭제
- [ ] 이미지 메타 데이터 및 댓글 조회
- [ ] 이미지 검색
- [X] 이미지 저장(내 보드에 추가)
- [ ] 이미지 상세 조회 → 댓글, 메타데이터, 현재 이미지와 비슷한 추천 이미지들
- [ ] 추천 이미지 조회 → 카테고리, 최근 본 이미지와 비슷한 이미지들
- [ ] 댓글 생성
- [ ] 댓글 삭제
